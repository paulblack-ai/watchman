1. `async def scrape_url(url: str, timeout: float = 15.0) -> str | None`:
   - Use `httpx.AsyncClient` with `timeout`, `follow_redirects=True`, and a User-Agent header `"Watchman/0.1"`
   - Fetch the URL, call `response.raise_for_status()`
   - Pass `response.text` to `trafilatura.extract(response.text, include_comments=False, include_tables=True)`
   - Return extracted text, or None if trafilatura returns empty/None
   - Catch all exceptions: log warning with URL and return None (graceful fallback per CONTEXT.md)
   - Log info on success with content length

Create `src/watchman/enrichment/extractor.py`:
1. `def _build_enrichment_prompt(card_title: str, card_url: str, card_summary: str | None, page_content: str | None) -> str`:
   - Build prompt instructing Claude to extract structured tool information for IcebreakerAI
   - Include card title, URL, and summary in the prompt
   - If page_content is provided, include it as the primary source
   - If page_content is None, instruct Claude to extract what it can from title, summary, and URL only
   - Instruct Claude: extract factual information ONLY, use null/empty list for fields not determinable from content, do NOT infer or fabricate pricing/capabilities not mentioned
   - Do NOT include any scoring data in the prompt (per CONTEXT.md: enrichment extracts facts independently)

2. `async def enrich_card(card_title: str, card_url: str, card_summary: str | None, page_content: str | None) -> IcebreakerToolEntry`:
   - Create `anthropic.Anthropic()` client (uses ANTHROPIC_API_KEY env var)
   - Call `client.messages.create()` with:
     - `model="claude-sonnet-4-20250514"` (Claude Sonnet per CONTEXT.md decision)
     - `max_tokens=1024`
     - `betas=["output-128k-2025-02-19"]`
     - `output_config` with `json_schema` format using `IcebreakerToolEntry.model_json_schema()`
   - Parse response with `IcebreakerToolEntry.model_validate_json(response.content[0].text)`
   - Set `source_url = card_url` and `discovered_at = datetime.utcnow()` on the result (overwrite LLM values to ensure accuracy)
   - Return the validated entry
