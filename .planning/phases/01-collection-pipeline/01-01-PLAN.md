
    **src/watchman/config/loader.py:**
    - `load_sources(config_path: Path) -> SourceRegistry`: open file, yaml.safe_load, SourceRegistry.model_validate. Raise clear error if file not found or validation fails.
    - `get_enabled_sources(registry: SourceRegistry) -> list[SourceConfig]`: filter enabled sources.
    - `parse_interval(freq: str) -> dict`: parse "4h"/"12h"/"1d" into {"hours": 4} etc for APScheduler.

    **src/watchman/storage/database.py:**
    - `init_db(db_path: Path) -> None`: async function that creates the database, enables WAL mode, and creates all tables:
      - `raw_items`: id, source_name, collector_type, title, url, summary, published_date, fetched_at, raw_data, processed
      - `cards`: id, title, source_name, date, url, tier, summary, collector_type, url_hash (UNIQUE), content_fingerprint, duplicate_of (FK to cards.id), seen_count, created_at
      - `source_health`: id, source_name, run_at, items_found, error_message, consecutive_zeros
    - Create indexes: idx_cards_url_hash, idx_cards_date, idx_cards_created_at, idx_raw_items_processed, idx_source_health_name
    - `get_connection(db_path: Path) -> aiosqlite.Connection`: context manager for getting a connection with row_factory set

    **src/watchman/storage/repositories.py:**
    - `RawItemRepository`: insert(item: RawItem) -> int, find_unprocessed() -> list[RawItem], mark_processed(id: int) -> None
    - `CardRepository`: insert(card: SignalCard) -> int, find_by_url_hash(url_hash: str) -> SignalCard | None, find_since(cutoff: datetime) -> list[SignalCard], increment_seen_count(card_id: int) -> None, link_duplicate(duplicate_id: int, canonical_id: int) -> None
    - `HealthRepository`: record_run(source_name: str, items_found: int, error: str | None) -> int, get_consecutive_zeros(source_name: str) -> int, reset_consecutive_zeros(source_name: str) -> None
    - All repositories take aiosqlite.Connection in constructor.
    - Use parameterized queries (never string interpolation) for SQL injection prevention.
