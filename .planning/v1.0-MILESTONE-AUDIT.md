## Resolved Gaps (from initial audit)

All three critical gaps identified in the initial v1.0 audit have been resolved by Phase 7:

### 1. Collector Registration Import Gap — RESOLVED
**Fix:** `jobs.py` changed `from watchman.collectors.base import get_collector` to `from watchman.collectors import get_collector`
**Evidence:** 6 new tests in `test_collector_registry.py` confirm COLLECTOR_REGISTRY has 3 entries (rss, api, scrape)

### 2. Naive Datetime in Model Defaults — RESOLVED
**Fix:** `raw_item.py` and `signal_card.py` changed `datetime.utcnow` to `datetime.now(timezone.utc)`
**Evidence:** Both model instances produce timezone-aware UTC datetimes; 86 tests pass

### 3. Gate 2 Timestamp Set Too Early — RESOLVED
**Fix:** `repositories.py` `set_gate2_state()` now only sets `gate2_reviewed_at` for non-pending states
**Evidence:** `test_gate2_pending_does_not_set_reviewed_at` confirms pending state does not set timestamp
